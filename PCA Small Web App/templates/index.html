What does this tool do?
Performs Principal Component Analysis (PCA) on your CSV data and generates a comprehensive PDF report with visualizations, loadings, and variable importance rankings.
Troubleshooting
Common issues include:
  1. Data Processing Error: Check that your CSV file contains numeric data. Columns with text or only missing values will cause errors. Non-numeric headers are allowed.
  2. PCA Calculation Error: This can happen if your dataset has too few rows (less than 2) or if a column has no variance (all values are the same).
  3. File Format: Ensure your file is a standard, comma-separated CSV.
